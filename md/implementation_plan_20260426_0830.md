# 籌碼數據「判斷說明」實作計畫

此計畫會在每個籌碼面數據框（如融資餘額、融資趨勢、三大法人等）的右上角加入「判斷說明」按鈕，使用者點擊後會彈出視窗（Bootstrap Modal）顯示詳細說明。

## Proposed Changes

### 1. 修改 CSS 樣式 (`index.html`)
- 在 `<style>` 中為 `.chips-stat` 加上 `position: relative;`。
- 新增一個 `.explanation-btn` 樣式，將按鈕固定在框的右上角，並加上點擊的 hover 效果。

### 2. 新增彈出視窗結構 (`index.html`)
- 在 `<body>` 底部新增一個 Bootstrap Modal 的 HTML 結構，用來顯示說明文字的標題與內容。
- 新增 `showExplanation(type)` 的 JavaScript 函數，負責替換視窗內容並彈出顯示。

### 3. 在籌碼區塊中加入標籤 (`index.html` 中的 `renderChipsSection`)
- 在「融資餘額」、「融資趨勢」、「外資」、「投信」、「自營商」五個卡片區塊中，分別加入 `<span class="explanation-btn" onclick="showExplanation('...')"><i class="bi bi-question-circle"></i> 判斷說明</span>`。

## 說明文字內容 (請您確認)

目前規劃的說明文字如下（融資趨勢已使用您提供的版本，其他項目為我補充的預設內容，如果不合適您可以隨時修改）：

1. **融資趨勢**：
   * **持續減少（籌碼安定）**：代表散戶這幾天正在連續賣出股票或停損出場。這些散戶賣掉的股票通常是被法人（外資或投信）接走了。散戶減少代表「籌碼變得乾淨安定」，未來股價上漲的阻力較小，被視為偏多的正面訊號。
   * **連續增加（散戶浮額增）**：代表散戶這幾天正在大量借錢買進這檔股票。如果這時候股價沒有跟著明顯上漲，代表法人可能在倒貨給散戶接盤。一旦股價下跌，這些借錢的散戶很容易因為恐慌而「多殺多」，風險較高，被視為偏空的負面訊號。
   * **持平**：代表近5天內融資沒有明顯連續增加或減少的趨勢，散戶動向不明顯。
2. **融資餘額**：
   * 融資餘額代表散戶借錢買進該股票的總張數。餘額越高代表散戶看多，但也代表潛在的賣壓（浮額）較重。若融資在高檔且股價不漲，需小心多殺多；若融資減少且股價上漲，代表籌碼流向法人，為偏多訊號。
3. **外資買賣超**：
   * 外資通常具有資金龐大、操作週期較長的特性。外資連續買超（連買大於3天）通常代表看好該公司基本面或整體趨勢，為強烈的偏多訊號；反之連續賣超則需避開。
4. **投信買賣超**：
   * 投信（國內基金）操作多追求波段績效，且有季底/年底作帳的特性。投信連續買超（特別是中小型股）通常代表會有波段行情，是尋找飆股與跟單的重要參考指標。
5. **自營商買賣超**：
   * 自營商（券商自有資金）操作偏向極短線，多為避險或短線套利。單看自營商的買賣超參考價值相對較低，但若三大法人同步買超（法人共振），則代表極強的多方共識。
